Shapes — shadow → highlight

Kies per slot een ingebouwde shape, of upload een eigen SVG met ⬆ (blauw = eigen SVG actief). Slot 1 = donkerste, slot 7 = lichtste. De preset hierboven wisselt alleen shapes/kleuren — alle andere settings blijven staan.

Artwork export

Kies een A-formaat hierboven én zet Formaat (Source) op A staand/liggend voor exact passend drukwerk. De SVG krijgt dan echte mm-afmetingen mee. Bij video pakt de export het huidige frame.

Hero export

Zonder URL wordt je huidige afbeelding in het bestand ge-embed — direct te openen, geen hosting nodig. Met URL blijft het bestand klein. HTML = drop-in standalone page. JSON = paste back to Claude voor een <DitherHero> React component.
